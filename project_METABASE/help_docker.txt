-------- HELP DOCKER COMPOSE (FLASK + METABASE) ---------------
Build and start Docker Compose:
docker-compose -p project_metabase up --build -d

Restart the Docker Compose:
docker-compose -p project_metabase start

Stop the Docker Compose:
docker-compose -p project_metabase stop

Remove the Docker Compose setup and associated containers:
docker-compose -p project_metabase down

------ HELP DOCKER METABASE ONLY (if you want to run Flask locally) --------
Run the Metabase Docker container:
docker run -d -p 3000:3000 --name metabase metabase/metabase

Restart the existing container:
docker start metabase

Stop the container:
docker stop metabase

Remove the container:
docker rm metabase
