------ Dans le flask project_ETL ------
Construire l'image:
docker build -t project_etl .

Lancer le docker en détaché avec 15g mémoire:
docker run -d -p 5000:5000 --name project_etl --memory="15g" project_etl
Lancer le docker en détaché:
docker run -d -p 5000:5000 --name project_etl project_etl
Lancer le docker en voyant les logs:
docker run -p 5000:5000 --name project_etl project_etl

redemarrer le docker existant:
docker start project_etl

arreter le docker:
docker stop project_etl
docker rm project_etl

Warning! The Docker image causes a +2 hour shift, so a scheduler set for 10:00 French local time in the code will run at 12:00 French local time inside the container.

---- Paramètrage WSL dans Windows -----
parametrer  15gb de mémoire et 4 processeurs pour wsl lorsque docker desktop est allumé en créant le fichier C:\Users\gaeta\.wslconfig
contenu du fichier:
[wsl2]
memory=15GB
processors=4
swap=0

redemarrer wsl avec PowerShell: wsl --shutdown

---- Activer / Desactiver authentification MongoDB ----
aller sur:
"C:\Program Files\MongoDB\Server\8.0\bin\mongod.cfg"

- Activer l'authentification, ecrire:
security:
  authorization: "enabled"

- Desactiver l'authentification, ecrire:
security:
  authorization: "disabled"

relancer MongoDB dans le terminal Powershell Administrateur:
net stop MongoDB
net start MongoDB
